# jsdoc-one-page

A one page jsdoc template based on the default jsdoc template.

## Getting Started

- npm install jsdoc
- npm install jsdoc-one-page
- jsdoc -t PATH/TO/jsdoc-one-page -r JS_SOURCE_DIR/FILE -d OUTPUT_DIR
